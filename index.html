1. 앱과 웹의 차이점은 무엇일까요?
제가 생각하는 앱과 웹의 가장 큰 차이점은, 앱은 이용하거나 업데이트를 하려면 사용자가 직접 스토어에 들어가 앱을 업데이트시키거나 다운로드를 따로 받아야 하지만 웹은 고쳐진 부분이 반영되기만 한다면 사용자가 웹사이트에 접속하거나 새로고침 시에 바로 업데이트된 페이지를 볼 수 있다는 것입니다.
2. 웹표준, 웹접근성, 시멘틱웹에 대하여 설명하세요.
웹표준은 브라우저의 종류나 버전에 따라 다른 기능 차이에도 호환이 가능하도록 제시된 표준이며, 공통요소를 사용하여 웹페이지를 제작하는 기법을 말합니다. 웹접근성은 장애인이나 고령자도 비장애인과 같이 동등하게 웹사이트에 접근하여 제공하는 정보나 기능을 이해하고 이용할 수 있도록 하는 것을 말합니다. 마지막으로 시멘틱웹이란 방대한 정보 속에서 컴퓨터가 스스로 데이터를 읽고 분석하고 분류, 가공할 수 있도록 하는 것입니다. 이는 웹상의 검색엔진의 최적화(SEO)를 위해서 나타난 용어이며 우리가 제공하는 데이터가 사용자의 검색에 따라 검색엔진에 잘 노출 되어야 하기 때문에 시멘틱 웹을 따르고 시멘틱 태그를 사용한 코딩을 지향하는 것입니다.
3. 자바스크립트 라이브러리 사용 시 가장 큰 장점은?
특정 코드를 따로 빼내어 함수(기능)가 반복적으로 들어가는 여러 곳에 쓸 수 있기 때문에 유지보수가 용이합니다. 또 일하는 시간을 단축시키고 좀 더 효율적으로 작업을 할 수 있다고 생각합니다.
4. 플러그인(Plugin) 이란 무엇인가요?
플러그인이란 컴퓨터의 프로그램이나 소프트웨어에 일부 기능을 추가해주는 한 부분으로, 보다 편하게 프로그램을 사용하고 추가적인 기능을 이용할 수 있도록 해주는 것입니다. 자세한 예시를 들자면 'vscode-styled-components'라는 플러그인은 설치하면 리액트에서 style을 줄 때 요소들이 자동완성이 되도록 해주는데 이는 일을 단순화 시킬 뿐 아니라 오타를 줄여줌으로서 오류를 범하지 않도록 도와줍니다. 결론적으로 비교적 단순한 작업에 많은 시간을 절약할 수 있습니다.
5. CSS 미디어쿼리는 어떤 역할을 수행하는지 답하시오.
미디어쿼리는 뷰포트의 크기에 따라 주고싶은 디자인을 선택적, 부분적으로 적용시킬 수 있게 해줍니다. 따라서 반응형 웹을 만드는 데에 매우 중요한 역할을 합니다.
